1. Keymap Menu                                  *keymap-menu.nvim-keymap-menu*

A lua plugin for Neovim that displays a searchable menu of key bindings that
can be executed.


FEATURES                               *keymap-menu.nvim-keymap-menu-features*

- search keybindings by lhs
- search keybindings by description
- includes:
    - default keybinds found in index.txt
    - overrides set via vim.keymap.set
